SOG - NEW GAMEPLAY RELEASE IS HERE!

The hard work of our developers during these last weeks bears its fruit! The new gameplay release is here!
What's new?

  • NEW SERVER
    The major news brought by this build is our new server! Maybe you are not as excited about it as we
    are, but, trust me, you should
    Our new server is more robust and will serve us a lot in the future. We've transferred several game
    fonctionalities from the application to the server side. This means that all the so far known cheats,
    as for example playing simultaneously on several devices, are no more possible!
  • NEW SPELLS SYSTEM
    Another big change, and this time you'll notice it very quickly, is a new spells system. Each character
    having a spell will release it at their first turn. The spell will stay on the playground, until you'll decide
    to trigger it by shooting it. Adding another player-depending factor means more strategy decisions
    during each turn.
    And now enemies can trigger spells too!
  • NEW LEVELS AND MORE
    You will discover the animations have changed, some obstacles are more complicated and there are two
    new levels too!

The testing Season 4 will start later this week. We want to be sure that the server transfer does not generate
any major issues, in order to avoid making any big changes during the season. We will bring you more news
shortly.

Please, note that all your achievements (except your blockchain based cards, of course) will be reset again
before the new season starts. Also, the leaderboard will be updated only with the beginning of the testing
season.

The Android users can download the new version here, the iOS users will receive an automated notification
shortly.

WARM UP TO BE READY FOR THE SEASON 4!
